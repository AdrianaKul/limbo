Detailed Description

template<typename Params, typename Model>
class limbo::experimental::model::GPParego< Params, Model >

this is the model used in Parego reference: Knowles, J. (2006). ParEGO: A hybrid algorithm with on-line landscape approximation for expensive multiobjective optimization problems. IEEE Transactions On Evolutionary Computation, 10(1), 50-66. Main idea:

  • this models aggregates all the objective values with the Tchebycheff distance
  • objectives are weighted using a random vector
  • a single model is built
